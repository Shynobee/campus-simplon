Create color blocks :

créer une mini application contenant :
Un champs input number,
un champs input color et
un bouton de validation.
Au click sur le bouton, on créé autant de div(s) qu'indiqué dans le champs number (1 par défaut). Le(s) block(s) créé(s) aura(ont) la couleur indiquée dans le champs color

Indices :

  • document.createElement
  • element.appendChild
  • element.value
  • element.style
  • element.onclick
